O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Ela define a estrutura e o significado do conteúdo da web.

Elementos Básicos

Os principais elementos estruturais do HTML incluem tags como <html>, <head>, <body>, títulos (<h1> a <h6>), parágrafos (<p>) e links (<a>).
